Public REST APIs
Free Tier
GitHub REST API
REST
https://api.github.com
Repos, issues, PRs, users, actions workflows. 60 req/hr unauthenticated, 5000/hr with token. Essential for CI/CD integrations and automations.
Docker Hub API v2
REST
https://hub.docker.com/v2
Image metadata, tags, manifests and layer info. Rate limited per IP. Authenticate via Bearer token for private repos.

Fake REST API for testing and prototyping. Full CRUD support — GET/POST/PUT/PATCH/DELETE. No auth needed. 100+ sample records.
OpenWeather API
Key Needed
https://api.openweathermap.org/data/2.5
Current weather, 5-day forecasts, historical data. Free tier: 60 calls/min, 1M calls/month. Useful for infra environment monitoring.
ipify — Public IP API
Free
https://api.ipify.org?format=json
Returns your public IP in JSON/text. No auth, no rate limit. Useful in scripts to detect NAT IPs, firewall whitelisting, and dynamic DNS updates.

DevOps & Infra APIs
Ops Grade
Kubernetes API Server
K8s
https://<cluster>:6443/api/v1
Full cluster control — pods, services, deployments, namespaces. Auth via ServiceAccount token or kubeconfig. kubectl is a REST client under the hood.
Prometheus HTTP API
Metrics
http://localhost:9090/api/v1
Instant/range queries, targets, alerts, rules. Used for dashboards and alerting pipelines. Add nginx basic auth or OAuth proxy in production.

Read/write secrets, manage policies and tokens. Auth via AppRole, AWS IAM, Kubernetes SA. Industry standard secrets management.
Grafana HTTP API

Trigger runs, manage workspaces, variables and state. Free tier includes 500 managed resources. Integrate with GitHub webhooks for GitOps.
// 02 — Authentication Methods
🔐
Token & Web Auth
Stateless
🎟️
JWT — JSON Web Token
Self-contained tokens signed with HMAC-SHA256 or RSA. Format: Header.Payload.Signature. Stateless — zero DB lookup. Used in Kubernetes, Vault, Firebase and most modern APIs. Expiry via exp claim.
# Send JWT in Authorization header
Authorization: Bearer eyJhbGciOiJIUzI1NiJ9...
🔑
API Key Authentication
Static keys via header or query param. Simple but requires rotation policy. Use X-API-Key or Authorization: ApiKey. Ideal for machine-to-machine calls where OAuth overhead is unnecessary.
sk-proj-xK9mL3nPqR8vT2wY5zQ7aB1cD4eF6gH...
🔒
OAuth 2.0 / OIDC
Industry standard delegated authorization. Four flows: Authorization Code, Client Credentials, Device, PKCE. OIDC adds identity on top. Powers GitHub Actions OIDC, GCP Workload Identity, AWS IRSA, and enterprise SSO (Okta, Azure AD, Keycloak).
🍪
Session + Cookie Auth
Server stores session state, client receives signed cookie. Simpler for browser apps. Use HttpOnly; Secure; SameSite=Strict flags. Requires sticky sessions or shared Redis in clustered deployments.
🛡️
Infrastructure Auth
Infra
🖧
SSH Key Authentication
Asymmetric key pairs for server access. Public key goes in ~/.ssh/authorized_keys. Prefer Ed25519 over RSA. Use ssh-agent and ForwardAgent in CI pipelines.
# Generate Ed25519 key pair
ssh-keygen -t ed25519 -C "devops@ci"
ssh-copy-id -i ~/.ssh/id_ed25519.pub user@host
☁️
AWS IAM / IRSA
IAM Roles for Service Accounts — attach AWS roles to K8s pods without static credentials. Temporary STS tokens auto-rotate every 15min. Works with EKS, GitHub Actions OIDC, Terraform Cloud, and cross-account access patterns.
📜
mTLS — Mutual TLS
Both client and server present X.509 certificates. Zero-trust standard for service mesh (Istio, Linkerd). Managed via cert-manager + Let's Encrypt or internal CA. Enforced at L4 — invisible to application code.
🤖
Vault AppRole
Machine-to-machine auth for CI/CD. App presents RoleID + SecretID to obtain a short-lived Vault token. SecretID can be wrapped (response-wrapping) for one-time delivery. Principle of least privilege built-in.
# Authenticate with AppRole
curl -X POST $VAULT_ADDR/v1/auth/approle/login
-d '{"role_id":"...","secret_id":"..."}'
